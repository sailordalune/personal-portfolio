# Z.Nasywa Portfolio

**Z.Nasywa Portfolio** adalah website personal portfolio milik **Zamira Nasywa Udhata**, mahasiswa Sistem Informasi yang memiliki ketertarikan pada bidang **UI/UX Design**, **Product Design**, **Front-End Development**, **Graphic Design**, **Digital Content**, dan dokumentasi kreatif.

Website ini dibuat sebagai media personal branding digital untuk menampilkan profil, keterampilan, pengalaman organisasi, sertifikasi, karya terpilih, serta kontak dalam satu website yang responsif dan mudah diakses.

## Website

Website dapat diakses melalui:

https://znasywa.vercel.app/

## Deskripsi Proyek

Website ini dikembangkan sebagai personal portfolio yang menampilkan identitas, kemampuan, pengalaman, dan karya dalam bentuk digital. Selain berfungsi sebagai media personal branding, website ini juga menjadi bentuk penerapan prinsip desain antarmuka pengguna dan pengalaman pengguna.

Secara visual, website ini menggunakan gaya desain yang playful, kreatif, dan ekspresif melalui penggunaan bold outline, card-based layout, warna yang cerah, serta elemen interaktif. Meskipun tampilannya cukup berani, struktur konten tetap dibuat rapi agar pengunjung dapat memahami informasi dengan mudah.

Proyek ini juga dibuat untuk memenuhi tugas mata kuliah Desain Antarmuka Pengguna dengan memperhatikan beberapa prinsip UI/UX, seperti konsistensi desain, navigasi yang jelas, hierarki visual, layout responsif, serta keterbacaan informasi.

## Fitur Utama

Website ini memiliki beberapa fitur utama, yaitu:

* Hero section dengan animasi role/title
* About Me dan Quick Bio
* Experience & Skills
* Availability Status
* Certifications
* Organization & Activities
* Selected Works / Project Gallery
* Filter kategori project
* Sorting project berdasarkan urutan terbaru atau terlama
* Preview Work pada project card
* Interactive cards seperti Play with Me, Create with Me, dan Learn with Me
* Halaman Contact berisi social links dan platform digital
* Tampilan responsif untuk desktop, tablet, dan mobile

## Cara Menjalankan Project

Pastikan perangkat sudah memiliki **Node.js** dan **pnpm**.

### 1. Clone repository

```bash
git clone https://github.com/sailordalune/Personal_portfolio.git
```

### 2. Masuk ke folder project

```bash
cd Personal_portfolio
```

### 3. Install dependencies

```bash
pnpm install
```

### 4. Jalankan development server

```bash
pnpm dev
```

Setelah server berjalan, buka browser dan akses:

```text
http://localhost:3000
```

## Penerapan UI/UX

Website ini menerapkan beberapa prinsip UI/UX, yaitu:

### 1. Konsistensi Desain

Komponen seperti card, button, tag, section title, dan layout digunakan secara konsisten di berbagai bagian website. Hal ini membantu tampilan website terlihat rapi dan mudah dipahami.

### 2. Navigasi yang Jelas

Navigasi utama terdiri dari halaman Home, Projects, dan Contact. Struktur navigasi dibuat sederhana agar pengguna dapat berpindah halaman dengan mudah.

### 3. Hierarki Visual

Penggunaan heading, warna, card, dan spacing membantu membedakan informasi utama dan pendukung. Dengan begitu, pengunjung dapat memahami isi website secara bertahap.

### 4. Keterbacaan Informasi

Ukuran teks, kontras warna, dan jarak antar elemen disesuaikan agar informasi tetap mudah dibaca pada berbagai ukuran layar.

### 5. Responsivitas

Layout website disesuaikan untuk perangkat desktop, tablet, dan mobile. Elemen seperti navbar, project card, activity card, dan contact card tetap dapat digunakan dengan nyaman pada layar kecil.

### 6. Kontrol Pengguna

Pada halaman Projects, pengguna dapat menggunakan filter kategori dan sorting untuk menelusuri karya sesuai kebutuhan.
